В файловой системе необходимо создать вручную файл содержащий пример
спектакля Ромео и Джульета.
При помощи потоков необходимо разбить данный файл на два, в каждом из которых
будет находится роль одного из актеров.

Пример исходного файла

Romeo: text for Romeo
Text for Romeo
Text for Romeo
Gulieta: text fo Gulieta
text fo Gulieta
text fo Gulieta
text fo Gulieta
Romeo: text for Romeo
